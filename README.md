# 🌴 Kerala Tourism Website

---

A complete tourism website showcasing **all 14 districts of Kerala**, highlighting their natural beauty, rich culture, and major tourist attractions.
This project is built using **HTML, CSS, and Bootstrap**, focusing on simplicity, responsiveness, and a smooth user experience.

---

## 🚀 Live Demo

🔗 https://treesa-prog.github.io/kerala-tourism-website/

---

## 📌 Features

* 🌍 Covers all **14 districts of Kerala**
* 🏝️ Dedicated pages for each district
* 🗺️ Integrated Google Maps for location view
* 🖼️ Tourist attraction cards with images and descriptions
* 📱 Fully responsive design using Bootstrap
* 🎨 Clean and modern user interface
* ⚡ Fully functional website

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5

---

## 📍 District Coverage

This website includes:

**Thiruvananthapuram • Kollam • Pathanamthitta • Alappuzha • Kottayam • Idukki • Ernakulam • Thrissur • Palakkad • Malappuram • Kozhikode • Wayanad • Kannur • Kasaragod**
